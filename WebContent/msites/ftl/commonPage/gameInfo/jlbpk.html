游戏说明Game Rule 加勒比扑克
游戏玩法 GAME INSTRUCTION
游戏说明

加勒比扑克(Caribbean Poker) 是以5张牌的牌面组合决定胜负的游戏。押底注后进行发牌，再根据手中5张牌的价值，决定是否与庄家的5张牌进行比较分胜负。
加勒比扑克下注方法简单，单纯比较牌型大小而不需考虑花色，规则简明且无太多额外限制，是一款容易上手的游戏。

游戏玩法
  1. 下底注后，荷官分别一次发给闲家与庄家各5张牌。发牌后，可以根据手牌牌型选择弃牌或是加注，加注则可与庄家进行比牌定胜负，弃牌则进行下一局。
  2. 会员下底注时可同时选择下注Bonus，若闲家当局手牌组合符合Bonus牌型，不论输赢或庄家是否成局，可依组合获得高赔率Bonus分数。
游戏规则
  1. 游戏使用“1副”共52张扑克牌进行游戏，每局结束后重新开始进行下一局。
  2. 牌局开始，下底注后点击发牌。
  3. 荷官进行发牌。先一次发给闲家5张牌，由左至右依序开牌后，再一次发给庄家5张牌。庄家的牌只开最后1张(庄家桌上位置最右边的牌)。
  4. 闲家选择“加注”或是“弃牌”。选择弃牌，庄家开牌显示牌型，点击“重复下注”或“新局”开始下一局；若选择加注，则庄家进行开牌并与闲家进行比牌。闲家获胜可赢得牌型组合相对应的赔率彩金分数。
  5. 下注项目说明如下：
下注项目 注额 说明
底注
Ante		 最小1，最大1000。 基本下注项目，需符合单注最小及最大单位，若无下注则无法加注及下注Bonus。
加注
Call		 2倍底注分数。 加注分数固定为2倍的底注分数。

Bonus		 固定只能下1注，1注分数为10。 下注底注后可决定是否下注Bonus，只需点击一下即可下注，每局只能下1注。
比牌规则

会员加注后才进行比牌的动作，比牌的规则如下：

  1. 庄家成局条件最少需要1张A+1张K。若庄家手牌无任何组合，亦同时无A+K，此时庄家不成局，闲家获胜，退回加注，取得底注1赔1的彩金分数。
  2. 庄家手牌成局且闲家又获胜，则闲家取得底注1赔1的彩金分数及加注赔率彩金分数。
  3. Bonus下注分数无论是否有押中皆不退回，若有押中派彩时再给予相对赔率的Bonus分数。

牌面大小的规则如下：

  • 纸牌花色：不考虑。
  • 牌面数字由大至小：A，K，Q，J，10，9，8，7，6，5，4，3，2。
  • 牌型为“顺子”时，以各牌数字定胜负，数字较大者胜； 若“A，2，3，4，5”形成“顺子”时，“A”牌作最小数字计。
  • 牌型、数字皆相同，则比较剩下的散牌数字决胜负，若还相同则为和局，和局退回底注和加注的下注分数。

介面说明 GAME INTERFACE
游戏画面
  • 主游戏画面(Main Game Screen)

游戏功能 GAME FUNCTION
  • Bonus : 可见Bonus赔率表与下注Bonus
  • 底注(Ante) : 下注底注
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 加注(Call) : 追加下注，注额为2倍底注分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 发牌(Deal)/加注(Call)/重复下注(Rebet) : 新局开始/增加注额(注额为2倍底注分数)/重复上局下注筹码
  • 清除下注(Clear)/弃牌(Fold)/新局(New Game) : 清除桌面所有下注分数/放弃该局/重新选择下注筹码
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
