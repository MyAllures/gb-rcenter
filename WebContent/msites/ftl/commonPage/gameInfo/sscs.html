游戏说明Game Rule 圣兽传说
游戏玩法 GAME INSTRUCTION

圣兽传说为拉霸加小玛莉的组合, 游戏视窗分为两部分, 中间以拉霸方式呈现，外围是小玛莉。当外围的中奖图案与中间的中奖图案相同时，便赢得分数。若中间三个图案相同还可以获得拉霸图案的分数。押注一次四单位［下注的分数会平均分配于押注拉霸三格及拉霸连线各1单位］，最多能押到125注。

 
中奖方式 WINNING TYPE
一般中奖

外围［小玛莉］停止的图案与中间［拉霸］图案有相同，即可获得该图案的赔率，如果3个转轮出现特别连线种类(参考"拉霸中奖赔率表")，还可以获得拉霸图案的奖金，所有转轮的奖金累计即为中奖分数。

特殊中奖

特殊中奖外围［小玛莉］中特殊灯(白虎、朱雀、青龙)则会有送灯的游戏进行。

右中白虎: 将会送出2~6个奖灯(送灯不会中特殊灯) 。

下中朱雀: 将会送出3~8个奖灯(送灯不会中特殊灯) 。

左中青龙: 将会出现连续4个奖灯 (此4个连续灯不会再次中青龙) 。

拉霸中奖

中间三个转轮都出现相同图案或其它情形(其它情形请参考下方拉霸中奖赔率表)，即可获得连线之中奖分数。

彩金中奖

当中间［拉霸］出现三个玄武龟图案，且外围［小玛莉］也停到玄武龟图案，即获得 [彩金中奖赔率表] 对应的彩金比例。

游戏功能 GAME FUNCTION
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 押注(BET) : 每注为四单位，最多可押125注;押一次下五注
  • 开始(SPIN)：下注後按开始，启动转轮
  • 停止(STOP) : 拉霸三个停止按钮
  • 清除(CLEAR) : 清除所有押注
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
