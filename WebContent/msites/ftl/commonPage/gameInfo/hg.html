游戏说明
扑克游戏中，使用六副牌，2~10都按其面值计算其点数，J 为11、Q 为12、K 为13、A 则为14。游戏的胜负是比第三张牌，是否介于两张牌中，与牌的花色无关。

游戏每局只能压一个筹码，开始押注后，庄家会发出两张牌，如果两张牌点数相减大于一，就会产生间距，例如：2和4的间距是一，6 和Q的间距是五。

  1. 如果两张牌点数相减大于一，玩家可以提高赌注（加倍），也可以叫牌（不加注），按”加注”或”叫牌”按钮，会再发一张牌，如果第三张牌恰巧介于两张牌点数中间则玩家获胜，反之，如果与前两张中的其中一张点数相同，或是前两张点数之外则庄家获胜。
  2. 如果两张牌点数相减等于一，则此局和局（赌注退还，本局游戏结束）。
  3. 如果两张牌点数相同，要再抽一张牌，如果第三张与前两张牌点数相同，则玩家以11：1的比率获胜，否则为和局。
赔率说明

皇牌：两张牌间距为零且两张牌不相等

一对：两张牌点数相同

三条：三张牌点数相同

支付分数：未含押注分数

断线时，如果还未选择加注或叫牌，此局会以”叫牌”结算分数

游戏功能 GAME FUNCTION
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 清除下注(CLEAR BET) : 清除所有下注
  • 发牌(DEAL) : 开始牌局
  • 重覆下注(REBET) : 重覆上局押注
  • 叫牌(CALL) : 不加注，继续牌局
  • 加注(DOUBLE) : 加注，并继续牌局
  • 新游戏(NEW GAME) : 继续新的牌局
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 音效(SOUND) : 按下 钮，启动或停止音效
