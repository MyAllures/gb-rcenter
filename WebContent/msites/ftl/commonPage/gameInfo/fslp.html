游戏说明Game Rule 法式轮盘
简介 INTRODUCTION

我们提供的是法式轮盘，转轮有 37个槽，号码为0，1到36。玩法相当简单，玩家决定下注区之後，电脑随机开出一号码，若有中奖再依中奖区的赔率给分，若无中奖即判定玩家输掉此局。

 
游戏玩法 GAME INSTRUCTION
  1. 开始新局後，请玩家在桌面不同的方格内或方格的边界上押放筹码。
  2. 压启动（Spin）後，轮盘开始转动。当轮盘停止後，滚球落入0-36其中一个细沟，该细沟外测所显示的号码即为开球号码。
  3. 若有中奖，依中奖区的赔率给分；若无中奖，即判定玩家输掉此局。
 
掷球规则 THROW RULES
  1. 本游戏进行方式为最接近实境赌场之设置，若有发生特殊情形将依本公司公告之办法处理。
  2. 游戏进行中因线路问题造成游戏中断 :
    • (a)若圆球已落入号码框格内，则该局游戏视为有效并由系统进行开配。
    • (b)若圆球在旋转中尚未落入号码框格内，玩家中断联机，则由电脑自行随机开启一颗球号，做为此局开球号码。
  3. 本公司所提供之每局电脑画面游戏结果与历史赛果纪录仅供参考。
 
投注种类 BET TYPES

轮盘的投注方式有 16个种类，详细说明如下 :

名称 说明
直接注 您可以投注包括 0在内的任意一个数字。下注时将筹码放到那个数字的中心 (如8丶23或35)。
分注 您可以投注於两个号码之间的格线上 (如2和3，或13和16)。
街注 您可以投注三个号码。将筹码押在轮盘桌上有三个数字顶部边线上即可 (如7丶8和9)。
三数 您可以投注於 0, 1, 2或0, 2, 3的交接区域。
角注 您也可以将筹码放到四个数字方框交叉的那个角区以下注那四个数字 (如19丶20丶22和23)。
四个号码 您可将筹码放到0和3最顶部的交点处以下注数字 0, 1, 2 和3。
线注 您可将筹码下注到两行相交的最顶部的交点处 ， 您可以赌两个街注，也就是两行的六个数字。 (如28丶29丶30及31丶32丶33便组成一条Line)。
列注 桌上每列数字底端有三个2to1字样的格子，一共有三个 。(如：3丶6丶9丶12丶15丶18丶21丶24丶27丶30丶33丶36)。这种赌区不包括 0 ，若开0则通杀。
下注一打数字 您可以将筹码下到有”D 12“，”M 12 ”和”P 12”字样的赌区以同时下注 12个数字。 (分别为1-12丶13-24及25-36)。(若开0则通杀)。
红色 /黑色 投注中彩号码将为红色或是黑色的，共投注十八个数字。(若开0则通杀)。
双数 / 单数 投注中彩号码将为单数或是双数，共投注十八个数字。(若开0则通杀)。
低注 / 高注 投注中彩号码将算低 (1-18)或是算高(19-36)，共十八个数字。(若开0则通杀)。

Neighbor Bets(旁注) :

当玩家选中一个号码下Neighbor Bets时，代表他下了这个号码及其前後各两个号码(轮盘位置上左右的号码)，总共为5个号码，皆为直接注（单一号码）。需要5个筹码

名称 说明
直接注 若开球号为Neighbor Bets中的号码，其赔率为1:36 。
