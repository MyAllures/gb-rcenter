游戏说明Game Rule 百家乐大转轮
游戏玩法 GAME INSTRUCTION
游戏特色

「百家乐大转轮」是款极具独家设计性的转轮游戏，将传统百家乐增加变化，配合转轮呈现新的游戏方式，带给会员更耳目一新的娱乐感。

游戏规则

  • 大转轮由内盘与外盘组合而成。

    外盘显示为：庄丶闲丶和，共分为28等份。

    内盘显示为：A丶2丶3丶4丶5丶6丶7丶8丶9丶10丶J丶Q丶K丶对丶百家乐，共分为28等份。

  • 游戏采内盘顺时针转动，外盘逆时针转动，转轮停止後以开奖球落到的位置即为当局开奖依据。

  • 开奖方式以内外盘的开奖球停驻位置组合後作为开奖依据。

    例：内盘停驻位置为K，外盘停驻位置为庄，则开奖结果为庄及庄大。

  • FreeGame：内盘的球落在『百家乐』，即触发百家乐FreeGame转轮，中间的转盘会翻转过来变成FreeGame次数指针转盘。

    接着游戏会自动开始转动指针根据停止位置决定给予的FreeGame次数。

  • 计分方式：个别下注分数 x 奖项赔率

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 清除下注(CLEAR)/重复下注(REBET)：清除所有押注或是重复上次押注
  • 开始(SPIN)：下注後按开始，启动转轮
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
