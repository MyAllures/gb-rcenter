游戏玩法 GAME INSTRUCTION

依喜好来下注，奖金依押注方式，赔率最少2倍，最多为58倍。下注分数最多可以押到100分。假设选择比例为10:1，下注苹果15分，则下注金额为150元。若中奖时赢得分数为 15 x 3＝45分 (15为下注分数，3为赔率)，则赢得金额为45 x 10 = 450元 (45为赢得分数，10为选择比例)。灯号在游戏开始后按顺时针方向依次变亮并转动，最终停在某一格图案上，当灯号停在所押注的图案时，便赢得奖金，也可以选择重覆下注，重覆上局押注。

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 清除下注(CLEAR)/重复下注(REBET)：清除所有押注或是重复上次押注
  • 开始(SPIN)：下注後按开始，启动转轮
基本功能 BASIC FUNCTION
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 局号(SN) : 游戏局号
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 音效(SOUND) : 按下 钮，启动或停止音效
