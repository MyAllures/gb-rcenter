游戏玩法 GAME INSTRUCTION
游戏说明
  • 「开心消消乐」是一款新型态、新玩法的游戏，各种可爱小动物，趣味登场。画面精美，上手简单，跟着动物一起拿高分!
游戏特色
  • 可爱小动物在六边形的盘面出现，4个邻近相同的对象即可消除，有三个特殊标记、彩金标记以及一个奖金游戏。
分数计算方式
  • 每局总得分为押注分数*个数赔率/50

奖金标记
  • 盘面任意出现3个时，即可进入奖金游戏
  • 方式:让四个以上的同色动物，横竖相连即可消除达成任务，获取分数
特殊标记1
  • 随机转换成一般符号、转换完后的符号，场上该符号则一起消除
特殊标记2
  • 标记自身周围最少3个，最多6个符号一同消除
  • *Any代表每个符号都有机会出现
特殊标记3
  • 该横排，立即转变为该符号
  • *Any代表每个符号都有机会出现

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 选择押注(SELECT BET) : 共10注，1注50分
  • 满注(MAX BET) : 满注500分
  • 自动开始(AUTO)/停止自动开始(STOP AUTO) : 选择自动开始或是停止自动开始
  • 开始(SPIN) : 开始
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分，游戏中可以直接点选比例进行更换
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
