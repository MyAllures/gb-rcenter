游戏玩法 GAME INSTRUCTION
游戏说明
3D数字大转轮利用简单的数字组合进行下注，是常见的博彩游戏，多元的下注方式更是许多人必玩的选择。下注玩法共分6种，赔率依下注的方式而有所不同。

游戏特色
  1. 丰富多元的下注组合。
  2. FREE GAME免费游戏次数。
  3. BONUS下注金额赔率加乘。

免费联机游戏(Free Game)
进入FREE GAME可获得免费游戏次数，获胜组合依最后下注为主。

BONUS
进入BONUS可获得当次下注分数乘上显示赔率的奖金。

游戏功能 GAME FUNCTION
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分
  • 重复下注(REBET)：重复上局押注
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 清除下注(CLEAR ALL BET)：清除所有押注
  • 开始(SPIN)：下注後按开始，启动转轮
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
