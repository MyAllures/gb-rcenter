游戏说明

中国象棋流传千年深具东方文化色彩，每一步的运筹帷幄更是绝妙不可言语，象棋大转轮结合暗棋与象棋游戏特色。

下注玩法共分：各别下注、组合下注、吃棋下注三种。

游戏特色

1. 中奖区依暗棋规则，两兵相交依棋阶大小吃棋。

※ 在此无须隔棋，即可依棋阶大小吃棋。

※当两军皆为相同棋阶时则为和棋。

2. 触发奖金特区时，依象棋规则过河飞吃，被吃的棋为增加「个别区」中奖的棋子( 不计入「吃棋区」派彩)。

3. 触发奖金特区时，依象棋规则过河冲吃，被吃的棋为增加「个别区」中奖的棋子( 不计入「吃棋区」派彩)。

图示 说明

※当 停留在a、b、c、d任一位置时，将对对角的棋子发动攻击，被攻击的棋子即为个别区增加的奖项。

游戏功能 GAME FUNCTION
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 赢得分数(WIN) : 本局赢得分数
  • 下注分数(TOTAL BET) : 本局下注分数
  • 清除下注(CLEAR)/重复下注(REBET)：清除所有押注或是重复上次押注
  • 开始(SPIN)：下注後按开始，启动转轮
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
