游戏说明

超级7是一款不同于一般水果盘的超级水果盘，游戏方式为押注1~8条线，每线押注最大10分。

游戏中包含比倍游戏、FEVER GAME、FEVER POOL、分散、重转、锁定、全盘彩金等等，立体绚丽的旋转画面搭配多样的玩法，超级7绝对是水果盘游戏的最佳选择。

介面说明 GAME INTERFACE

游戏画面

  • (A 游戏画面) 图1、2、3、4、5、6、7、8：皆表示联机的位置。
  • (B 游戏信息) FEVER GAME图与：达成联机时数字会减1，减到0时，送免费游戏1次。
  • (C 游戏信息) FEVER POOL图与：达成联机时数字会减1，减到0时，送FEVER POOL彩金。
  • (D 游戏信息) ALL POOL图与ALL ELSE：出现全盘7，送ALL POOL彩金；出现其他全盘，送1/2 ALL POOL彩金。
 

游戏功能 GAME FUNCTION

  • 下注分数(TOTAL BET) : 本局下注分数
  • 每线押注(LINE BET) : 每线押注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分
  • 押注(BET)：选择押线数与每线押注分数
  • 全押(MAX BET)：直接押满8线，每线10分
  • 开始(SPIN)/停止(STOP) : 开始或是停止
  • AUTO / OFF：自动开始或停止自动开始
 

基本功能 BASIC FUNCTION

  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
