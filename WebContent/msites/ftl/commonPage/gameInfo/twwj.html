游戏特色

『特务危机』故事发生在30年代的上海滩，主角是身经百战绰号『神枪』的特务，在一次执行例行性情报搜集任务中，无意间察觉一件足以撼动全国的暗杀行动，而这个行动和原本应该保护国家安全为首要任务的特工局『军机长』，有着非比寻常的关联性；而神枪正隶属于这个特工局。接踵而来的是局内的背叛，以及同僚的暗杀与追击。神枪只能暂时躲避到他情人『艳云』的工作地点─上海知名的天堂夜总会。

为了帮助神枪揭穿军机长的阴谋诡计，艳云也从中进行协助。而神枪又如何能够躲避追杀，并在关键时刻反将一军呢？就在这彩金游戏里展开谍对谍的精彩对决吧！

游戏说明

以5个转轴搭配20条联机，包含累积式免费联机游戏、累积彩金(Jackpot)、倍数奖金、可变标记的3D拉霸游戏。奖金联机的计算方式是由左而右出现3个以上相同对象，依投注的额度和中奖标记获得相对应的奖金。

主要对象：机密文件、真伪莫辨、暗杀。

次要对象：神枪、艳云、军机长、特务枪战、情人相遇、天堂夜总会、回身射击、窃听、地图标记。

彩金游戏：真伪莫辨〈Fact or fiction〉、暗杀〈Assassination〉、免费联机游戏〈Free Game〉。

联机规则：

  • 4个Jackpot累积彩金：Mini/Minor/Major/Grand.
  • 最高900倍的奖金赔率。
  • 奖金标记、免费联机游戏、可变标记和彩金标记。
  • 每线取最高赔率为得分依据。
  • 得分由每条押注线的左边开始计算。
  • 每局总得分为每线押注乘上赔率。

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 每线押注(LINE BET) : 每线押的分数，最多达20分
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分，游戏中可以直接点选比例进行更换
  • 选择押线(SELECT LINES) : 投注押线，最多达20线
  • 满线满注(MAX BET) : 直接押满20线，每线20分
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 开始(SPIN)/停止(STOP) : 开始或是停止
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
