游戏说明Game Rule 猴子爬树
游戏玩法 GAME INSTRUCTION
游戏特色

「猴子爬树」是一款家喻户晓游戏。以六只逗趣的猴子为主题，在椰子树爬行的竞技游戏，若你能准确预测获胜的号码，高额彩金等着您抱回家。

游戏说明
  • 游戏开始选择押注1、5、50筹码，押注您预测会获胜的猴子号码组别。
  • 点击开始则进行比赛，比赛过程中会有香蕉、椰子、大嘴鸟增加游戏趣味，此时无法押注直到此局比赛结束为止。
  • 共14种赔率，单个押注上限200分。
  • 分数计算方式为=下注区分数X猴子赔率
道具说明
香蕉 说明

猴子短暂提升爬行速度

椰子 说明
让猴子延缓爬行速度
大嘴鸟 说明
让猴子延缓爬行速度

游戏功能 GAME FUNCTION
  • 赢得分数(WIN) : 本局赢得分数
  • 总押(TOTAL BET) : 本局押注的分数
  • 余额(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 开始（SPIN） : 下注后按开始，开始竞赛
  • 自动开始(AUTO PLAY)/自动取消(STOP AUTO PLAY) : 选择自动开始或是自动取消
  • 取消(CANCEL) : 清除所有押注
 
基本功能 BASIC FUNCTION
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 局号(SN) : 游戏局号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
