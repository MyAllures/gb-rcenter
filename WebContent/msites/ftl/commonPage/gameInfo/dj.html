游戏说明

游戏方式为拉霸9线：以5个转轴搭配双向的9条连线，结合了奖金标记，免费连线游戏(Free Game)，可变标记(Wild)，彩金标记(Jackpot)等的拉霸游戏。彩金连线的计算方式是由左或右出现2个以上相同物件，依投注分数和中奖标记获得相对应的彩金分数。

主要物件：免费游戏。

次要物件：冠军奖杯(Championship Trophy)、优胜奖牌(Medal)、喷水(Spray of water)、劲霸(Jin Bar)、斗鸡刀(Cockfighting daggers)、美金(Money)、攻击(Attack)、鬼子斗鸡(Black cock)、白马王(White cock)。

彩金游戏：斗鸡比赛。

联机规则：

  • 4个Jackpot累积彩金：Mini/Minor/Major/Grand.
  • 奖金标记、免费连线游戏、可变标记和彩金标记。
  • 每线取最高赔率为得分依据。
  • 得分由每条押注线的左边或右边开始计算。
  • 每局总得分为每线押注乘上赔率。

※获得左边连线美金标记时，即增加左方累积分数

※获得右边连线美金标记时，即增加右方累积分数


※获得左边连线攻击标记时，即左方斗鸡攻击右方斗鸡

※获得右边连线攻击标记时，即右方斗鸡攻击左方斗鸡

※攻击优先顺序为:1.标记数量 2.连线编号(1优先9最后)

※如同一线同时获得5个攻击标记，则此回合攻击双方平手


※左方斗鸡获胜时，可赢得左方斗鸡比赛累积分数 X 分数倍率的奖励

※右方斗鸡获胜时，可赢得右方斗鸡比赛累积分数 X 分数倍率的奖励

※累积分数倍率共4种：【x1】、【x2】、【x3】、【x5】

※斗鸡获胜后，累积分数倍率将会增加，最大可至【x5】，直至斗鸡被击倒为止

※斗鸡被击倒后，累积分数倍率将会回复为【x1】

※当进行洗分、断线或退出，累积分数、斗鸡位置、斗鸡比赛分数倍率不保留

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分，游戏中可以直接点选比例进行更换
  • 选择押线(SELECT LINES) : 投注押线，最多达9线
  • 左线押注(LEFT BET) : 增加左边9线押注分数
  • 开始(SPIN)/停止(STOP) : 开始或是停止
  • 右线押注(RIGHT BET) : 增加右边9线押注分数
  • 满线满注(MAX BET) : 直接押满左、右9线，每线20分
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
