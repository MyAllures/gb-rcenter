游戏说明

二十一点是一个广受欢迎的扑克牌游戏，10、J、Q、K点数都当作10点，Ace(A)可当作1点或11点，2到9则为原面值点数。 一开始将各发给玩家两张牌，玩家可以看到庄家其中一张牌，之后可根据自己的牌点数和庄家那张已翻开的牌，决定是否要补牌。牌值以求取得更接近21点，而又不超过21点的点数。点数越大且不超过21点为胜方。另外，"黑杰克"(最初两张牌面值21点)比总点数是21点的牌要大。如果您手上的牌比庄家的更接近21点，您将赢得等同于下注金额的钱。如果您得到黑杰克，您将赢得1.5倍下注金额的钱，除非恰巧庄家也中了黑杰克， 那么结果为和局，下注金额将全部退还。

庄家规则

庄家在继续要牌 "补牌" 或者维持目前的持牌时 "停牌"要遵循一定的规则。庄家拿牌或者停牌与您手上的牌为多少无关，他必须遵循如上的规定。作为玩家的您可以根据庄家手上的牌来决定拿牌或停牌。(庄家补牌条件请参考下面列表)

分拆

分拆成两付牌并补下与原下注金等额的下注金额。然后按照正常的步骤要牌。如果您“分拆”后点数达到21点，只能算21点，不算黑杰克。(分拆条件请参考下面列表)

加注

如果您感觉您将得到的下一张牌将极有可能击败庄家，那么您可以选择"加注"。于是您的下注金额将增加一倍，但是您只能再拿一张牌。(加注条件请参考下面列表)

保险

如果庄家得到的第一张牌是A，他将有相当大的可能中到黑杰克。此时您可以选择是否购买保险。如果您相信庄家的下一张牌会是10，您可以用等同于您下注金额的一半购买保险。 如果庄家确实中了黑杰克，保险将支付您双倍的保金也就是您当初下注的金额。如果您购买了保险但是庄家没有中到黑杰克，那么您将失去购买保险的金额。您最初的下注视您以及庄家得到的牌决定输赢。如果您和庄家都得到黑杰克，您的下注筹码将被退还。

游戏功能 GAME FUNCTION
  • 发牌 : 新局开始
  • 重覆下注 : 重覆上局押注额
  • 可用余额： 显示玩家目前可以游戏的金额
  • 拿牌 : 取补牌
  • 听牌 : 停止补牌
  • 加注 : 增加注额(注额值同下注金额)
  • 分牌 : 将同牌组拆为两副牌组
  • 保险 : 庄家第一张为Ace时，闲家加注赌庄家21点(注额为下注金额一半)
  • 清除下注 : 清除桌面所有下注金额
基本功能 BASIC FUNCTION
  • 会员帐号 : 显示玩家登入帐号
  • 局号： 游戏局数
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 历史纪录(HISTORY)： 按下 钮，下注纪录数据文件
  • 帮助(HELP)： 按下 钮，规则说明
  • 离开(EXIT)： 按下 钮，离开本游戏，跳大厅窗口
  • 音乐(MUSIC)： 按下 钮，启动或停止背景音乐
  • 音效(SOUND)： 按下 钮，启动或停止音效
