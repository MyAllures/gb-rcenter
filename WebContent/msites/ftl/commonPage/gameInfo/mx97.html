介面说明 GAME INTERFACE

游戏画面

  • 图 A、B、C、D、E、F、G、H :皆表示联机的位置
  • FEVER图与：表示出现结果为樱桃或铜钟3个联机时，数字会减1，当数字减到0时，就会有免费赠送一次，有机会开出大牌。
  • 图与：若出现全部7或全部蓝色BAR时，赠送彩金的分数
 

游戏功能 GAME FUNCTION

  • 下注分数(TOTAL BET) : 本局下注分数
  • 每线押注(LINE BET) : 每线押注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 全部停止(ALL STOP)： 滚轮开始后，停止所有滚轮
  • 开始(SPIN)：开始
 

基本功能 BASIC FUNCTION

  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
