游戏说明

空战英豪 拉霸游戏，是以五个转轴搭20条联机，包含免费联机游戏、累积彩金(Jackpot)、分散标记、可变标记的拉霸游戏。彩金联机的计算方式是由左而右，依投注的额度和中奖的标记获得对应的彩金。

20条联机 ：

  • 4个Jackpot累积彩金：Mini/Minor/Major/Grand.
  • 最高3000倍的奖金赔率。
  • 奖金标记、分散标记、可变标记和免费联机游戏。
  • 每线取最高赔率为得分依据。
  • 得分由每条押注线的左边开始计算。
  • 每局总得分为每线押注乘上赔率。

游戏功能 GAME FUNCTION
  • Balance : 显示玩家目前可以游戏的分数
  • Bet : 本局下注分数
  • Win : 本局赢得分数
  • BACK : 离开游戏
  • Lines Selected : 本局下注的线数
  • Bet Pre Line : 每线押的分数，最多达20分
  • CREDIT EXCHANGE : 可在此选择比例大小、开分和洗分
  • AUTOPLAY / STOPAUTO : 选择自动开始或是停止自动开始
  • LINES : 投注押线，最多达20线
  • SPIN / STOP : 开始或停止
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
