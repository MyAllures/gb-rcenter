游戏说明Game Rule 金刚爬楼
游戏玩法 GAME INSTRUCTION
游戏特色

以六只逗趣的金刚为主题，在大楼中爬行的竞技游戏，若你能准确预测获胜的号码，再加上独特的加倍机制，高额奖金等着你抱回家。

游戏说明
  • 玩家可自由切换筹码1,5,50以方便下注。
  • 点击开始则进行比赛，此时无法押注直到此局比赛结束为止。
  • 共6种赔率，单个押注上限为50分。
  • 分数计算方式为=下注区分数 X 金刚赔率 X 加倍 (加倍机制非100%触发)。
道具说明
飞行艇 说明

比赛中有机率会触发分数加倍的飞行艇，飞行艇所加倍数可能为两倍或三倍。飞行艇只会在玩家成功押对获胜号码时发挥作用。

游戏功能 GAME FUNCTION
  • 赢得分数(WIN) : 本局赢得分数
  • 总押(TOTAL BET) : 本局押注的分数
  • 余额(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 开始(SPIN) : 下注后按开始，开始竞赛
  • 自动开始(AUTO PLAY)/自动停止(STOP AUTO PLAY) : 选择自动开始或是自动停止
  • 清除下注(CLEAR ALL BETS) : 清除押注分数
基本功能 BASIC FUNCTION
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 局号(SN) : 游戏局号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
