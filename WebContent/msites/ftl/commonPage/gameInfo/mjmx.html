游戏玩法 GAME INSTRUCTION
游戏特色

突破传统的老虎机设计，「秘境冒险」是一款新型态的迷宫寻宝游戏。透过角色在丛林迷宫里的探索丶挖掘宝藏丶以及在遗迹神庙里的奇遇，让会员充分感受寻宝的乐趣，为自己获得大量的财富。

游戏说明

游戏是由3 x 5 共15格的迷宫路径所组成，并结合了彩金游戏(Bonus Game)丶免费游戏(Free Game)丶彩金标记 (Jackpot)等游戏。

游戏进行方式：

由左开始出现3格以上连续路径，角色向前行进至端点进行挖掘动作。挖掘後可获得宝箱或是出现地洞进入彩金游戏。形成特殊路径时可进入免费游戏或是获得彩金分数。

特殊物件：地洞，特殊路径。

一般物件：1~7号路径。

彩金游戏：寻找珍宝(FIND THE TREASURE)丶免费游戏(Free Game)。

宝箱分数计算说明
  • 得分由左开始，连续行进3格(含3格)以上开始计算。
  • 3格~12格所得宝箱设有固定赔率，12格以上以最大赔率200倍计算。
  • 押注分数：每个角色最少5分，最大25分，每局皆需押注3个角色。每回总押注分数=押注分数x 3(15~75)。
  • 角色所得分数=角色行走路径格数赔率x押注分数；每局总得分=各角色分数的总和。
  • 角色行走路径经过相同格数时，可重复计算格数。

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
  • 押注(SELECT BET) : 投注押注，有5种选项可供选择，最多为25
  • 满注(MAX BET) : 直接押满 25 x 3(角色) = 75
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 开始(SPIN)/停止(STOP) : 开始或是停止
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
