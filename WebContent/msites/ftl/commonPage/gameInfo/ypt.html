游戏说明Game Rule 玉蒲团
游戏玩法 GAME INSTRUCTION
游戏特色

「玉蒲团」是以明朝李渔写的章回艳情小说「肉蒲团」为基础改编而成，其内容荒诞不羁，且多以写实艳情描述。在中港台三地广负盛名，并改拍为多部电影，是煽情且肉欲的情色题材。

游戏说明

以5个转轴搭配20条联机，包含免费联机游戏、累积彩金(Jackpot)、奖金标记、可变标记、分散标记、特殊牌组的3D拉霸游戏。奖金联机的计算方式是由左而右出现3个以上相同对象，依投注的额度和中奖标记获得相对应的奖金。

主要对象：秘宝阁大门、密眼偷窥、华陀密经。

次要对象：未央生、冬梅、铁玉香、木马、玉葡萄、炼丹炉、红珊瑚、玉笔、折扇。

彩金游戏：九阳奇术〈Man Power Operation〉、欲池绮色〈Peeping Bathroom〉、免费联机游戏〈Free Game〉、绝叫欲马〈Sexy Definition〉。

联机规则：

  • 4个Jackpot累积彩金：Mini/Minor/Major/Grand.
  • 最高900倍的奖金赔率。
  • 奖金标记、免费联机游戏、分散标记、可变标记和彩金标记。
  • 每线取最高赔率为得分依据。
  • 得分由每条押注线的左边开始计算。
  • 每局总得分为每线押注乘上赔率。

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 每线押注(LINE BET) : 每线押的分数，最多达20分
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分，游戏中可以直接点选比例进行更换
  • 选择押线(SELECT LINES) : 投注押线，最多达20线
  • 满线满注(MAX BET) : 直接押满20线，每线20分
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO PLAY) : 选择自动开始或是停止自动开始
  • 开始(SPIN)/停止(STOP) : 开始或是停止
 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
