游戏玩法 GAME INSTRUCTION
游戏特色

「扑克拉霸」是以扑克牌为设计的单线拉霸游戏。

游戏说明

单线拉霸游戏，共有3个转轴，当转轴停止後，停在中奖线的图示符合中奖的图案，玩家就会获得中奖图案所表示的中奖分数。

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 押注(BET) : 每单位注额，固定为1。
  • 分数兑换(CREDIT EXCHANGE) : 选择比例大小、开分和洗分，游戏中可以直接点选比例进行更换
  • 三倍注(Bet Max) : 点击此按钮来下满投注分数。
  • 一倍注(Bet One) : 点击此按钮来增加投注分数，每次增加之投注分数为拉霸机之基本注额。
  • 自动开始(Auto Play)/停止(Stop) : 点选自动开始後，拉霸机会记录您最後一次之投注额。当您按开始後即进行游戏，直到按了停止则取消自动开始。
  • 开始(SPIN) : 当您下注後，可以点选开始来启动游戏之进行，当转轮停止後，您也可以按开始将会继续上一次的投注额再玩一次此游戏。
 
基本功能 BASIC FUNCTION
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 局号(SN) : 游戏局号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
