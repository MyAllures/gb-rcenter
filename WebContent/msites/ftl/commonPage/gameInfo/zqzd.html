游戏说明Game Rule 蒸气炸弹
游戏玩法 GAME INSTRUCTION
游戏说明

蒸气炸弹(POP BOMBER)是一款轻松活泼的消除游戏、配合各式各样道具，可以获得更多分数。

  • 主打符号连消，期待多重奖励。
  • 消除补上符号，特殊道具符号，累积拼图送分及无敌爆破模式。
  • 4个Jackpot累积彩金：Mini／Minor／Major／Grand.
  • 相同符号只要相邻三个以上即可消除。
  • 可同时消除多种符号。
  • 每个符号有各自赔率，消除越多分数越高！
  • 派彩分数=中奖颗数*单颗赔率*押注等级（1〜10）。

拼图送分（Pattern）

进入条件：只要消除的符号有对应图案图形的位置即可触发。

玩家累积至符合模式图形的框格收集完毕即可获得奖金分数：

收集完成赠送1〜3倍押注分数。

假设更换下注分数，拼图会被重新置换。
无敌爆破模式(Fever)

进入条件：同一局符号连续消除5次以上即可触发。
消除第5次后分数X2倍
消除第10次后分数X3倍

彩金标记(Jackpot)

牌组中出现3〜6个彩金标记即获得头奖彩金，派彩后结束此局。

Minor → 相邻4颗

Major → 相邻5颗

游戏功能 GAME FUNCTION
  • 下注分数(TOTAL BET) : 本局下注分数
  • 赢得分数(WIN) : 本局赢得分数
  • 可用分数(CREDITS) : 显示目前可以游戏的分数
  • 押注等级（BET LEVEL） ： 共等级1〜10，每个等级50分
  • 满注（MAX BET） ： 押注等级= 10
  • 自动开始(AUTO PLAY)/停止自动开始(STOP AUTO) : 选择自动开始或是停止自动开始
  • 开始（SPIN） ： 开始
  • 分数兑换(CREDIT EXCHANGE) : 可在此选择开分与洗分
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能
  • 音效(SOUND) : 按下 钮，启动或停止音效
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
