游戏说明Game Rule 趣味台球
游戏玩法 GAME INSTRUCTION
游戏特色

趣味台球是一款简单趣味的消除游戏、快速连续消除的玩法，加上特殊游戏，让玩家获得高额的分数。

游戏说明

在5*5的方形区域，随机派发相对应的台球，分别有1号球、3号球、6号球、10号球、12号球和1种特殊8号球。

相同台球只要相连5个以上即可消除获得分数，同时相连的台球消失，剩余台球依次下移之后，游戏自动补充台球，以此类推，直到当前无符合规则的相连台球为止。

派发后，若有8号球，先消除8号球，待游戏补上球之后，剩余台球再依规则消除。

 

分数计算方式为派彩分数=使用分数/10 X 台球赔率

特殊游戏
图片 说明

彩金说明

出现21个及以上相连的相同颜色台球时，获得Jackpot彩金，派彩后结束此局。
※Jackpot：彩金的该次消除，不会获得消除分数。
介面说明 GAME INTERFACE
游戏画面

  • 主游戏画面(Main Game Screen)

  • 特殊游戏

进入特殊游戏后，若15秒内玩家无点击「击球」，游戏将自动进行特殊游戏。

游戏功能 GAME FUNCTION
  • 账户游戏积分(CREDITS)：玩家剩余分数
  • 8号球(MAGIC 8-BALL)：显示目前累积进度
  • 菜单(MENU)：开启基本功能选单
  • 分数兑换(CREDIT EXCHANGE)：开分和洗分
  • 使用分数(BET)：调整下注分数
  • 开始游戏(START)：开始
  • 托管(AUTO)/取消托管( STOP AUTO)：自动开始或停止自动开始
历史纪录
  • 曲线为近10次玩家游玩的历程记录，最右边为最新一次纪录，反之为最旧。
  • 白色表示未中奖、黄色表示中奖分数小于1倍、橘色表示1倍≦分数≦3倍、红色表示分数>3倍。
  • 联机中断、重载或离开游戏曲线图将不保留纪录。

退出游戏
  • 选择「确定」可保留当前黑球累积数量。
  • 选择「取消」则返回游戏画面。

 
基本功能 BASIC FUNCTION
  • 局号(SN) : 游戏局号
  • 会员帐号(USER ID) : 显示会员登入帐号
  • 存款(DEPOSIT) : 按下 钮，开启会员中心的存款功能(此功能仅于PC版有)
  • 音乐(MUSIC) : 按下 钮，启动或停止背景音乐
  • 历史纪录(HISTORY) : 按下 钮，历史下注纪录
  • 帮助(HELP) : 按下 钮，规则说明
  • 离开(EXIT) : 按下 钮，离开本游戏，跳大厅窗口
